* Nmap.org

* Scanme.nmap.org

	* 네트워크 인터페이스 & 커널
	* RFC, 패킷 구조, 등 *

# Nmap 플래그 
-p 포트(22,80,443 | 100-200| --top-ports 100)
-iL  타겟 리스트 파일
-Pn 핑 제외 포트 스캐닝
-sV 네트워크 서비스 배너확인
-sC 기본 Nmap 정보 수집 스크립트 사용 
-oA 다양한 (xml, nmap, gnmap) 형태로 출력
-T0~ T5 속도
-n DNS Resolution 제외 
-- open  열려있는 포트만 출력 

# 포트스캐닝 종류 

* Host Discovery 
	* ICMP + 포트 80,443 확인
* TCP
	* -sS: Tcp Syn
	* -sT: TcP Connect
	* Ack, Null, Fin, X-Mas, ...: 현대 방화벽 때문에 x
* UDP
	* -sU: UDP 포트스캔
* ICMP (포트스캐닝 X)*
